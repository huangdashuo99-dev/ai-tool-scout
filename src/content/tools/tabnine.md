## Overview

Tabnine is an AI code completion tool that prioritizes privacy by running models locally on your machine. It provides intelligent code suggestions across multiple languages and IDEs while keeping your code on your own hardware.

## Key Features

- **Local AI Processing**: Code completions run on your machine — no code leaves your computer.
- **Multi-IDE Support**: Works in VS Code, JetBrains, Vim, Neovim, Emacs, and more.
- **Team Learning**: Optionally learns from your team's codebase for better suggestions.
- **Full Line Completion**: Predicts entire lines of code, not just partial suggestions.
- **Natural Language to Code**: Describe what you want and get code suggestions.

## Pros & Cons

### Pros

- Strong privacy — local processing option
- Fast completions with low latency
- Good multi-language support
- Team codebase learning option

### Cons

- Local model less powerful than cloud-based alternatives
- Advanced features require paid plan
- Free tier has limited completions per month
