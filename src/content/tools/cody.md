## Overview

Cody is Sourcegraph's AI coding assistant that stands out by understanding your entire codebase, not just the current file. It provides code suggestions, explanations, and chat powered by Sourcegraph's code intelligence platform.

## Key Features

- **Full Codebase Context**: Understands your entire repository for more accurate suggestions.
- **Code Chat**: Ask questions about your codebase and get answers with file references.
- **Autocomplete**: Multi-line code completions with deep context awareness.
- **Code Explanations**: Highlight code and get instant explanations.
- **Search & Repair**: Find and fix errors across your codebase.

## Pros & Cons

### Pros

- Completely free for individual developers
- Full codebase context is very powerful
- Good code search and navigation
- Open-source backend available

### Cons

- Fewer IDE options than competitors
- Less brand recognition
- Enterprise features require paid plan
