## Overview

Cursor is an AI-first code editor built on VS Code that deeply integrates artificial intelligence into every aspect of the coding workflow. Rather than being a VS Code extension, Cursor is a full standalone editor that replaces VS Code while maintaining full compatibility with all extensions, themes, settings, and keybindings.

What makes Cursor unique is how thoroughly AI is woven into the editing experience. From inline code generation and multi-file editing to a powerful chat interface that understands your entire codebase, Cursor is designed for developers who want AI to be a core part of their development process — not just an autocomplete plugin.

With rapid adoption and strong word-of-mouth, Cursor has emerged as one of the most exciting developer tools of 2025, offering a compelling alternative to the VS Code + Copilot combination.

## Key Features

- **Multi-Model AI Support**: Choose between Claude 3.5 Sonnet, GPT-4, and other models for different tasks — code generation, chat, and editing.
- **Codebase-Wide Chat**: Chat with your entire codebase — Cursor indexes your project and can answer questions about architecture, dependencies, and code logic.
- **Inline Editing (Cmd+K)**: Select code and describe changes in natural language. Cursor edits the code in place with streaming diffs you can review and accept.
- **Composer Mode**: A powerful agentic mode that can plan and execute multi-file changes, create new files, and handle complex coding tasks autonomously.
- **Terminal Integration**: AI-powered terminal that understands context and can suggest or execute shell commands related to your code.
- **Privacy Mode**: Run models locally or use privacy mode to ensure your code is never stored on external servers.

## 概述

Cursor 是一款基于 VS Code 构建的 AI 原生代码编辑器，将人工智能深度集成到编码工作流的方方面面。与 VS Code 扩展不同，Cursor 是一个完整的独立编辑器，可以替代 VS Code，同时完全兼容所有扩展、主题、设置和快捷键。

Cursor 的独特之处在于 AI 与编辑体验的深度融合。从内联代码生成和多文件编辑，到一个能理解你整个代码库的强大聊天界面，Cursor 专为那些希望 AI 成为开发流程核心部分 — 而不仅仅是一个自动补全插件 — 的开发者而设计。

凭借快速的用户增长和强大的口碑传播，Cursor 已成为 2025 年最令人兴奋的开发者工具之一，为 VS Code + Copilot 的组合提供了极具竞争力的替代方案。

## 核心功能

- **多模型 AI 支持** — 针对不同任务在 Claude 3.5 Sonnet、GPT-4 等模型之间选择 — 代码生成、对话和编辑
- **代码库级对话** — 与你的整个代码库对话 — Cursor 会索引你的项目，能回答关于架构、依赖和代码逻辑的问题
- **内联编辑（Cmd+K）** — 选中代码并用自然语言描述修改。Cursor 会就地编辑代码，以流式 diff 展示供你审核和接受
- **Composer 模式** — 强大的智能体模式，能规划和执行多文件变更、创建新文件，自主处理复杂编码任务
- **终端集成** — AI 驱动的终端，能理解上下文并建议或执行与代码相关的 shell 命令
- **隐私模式** — 本地运行模型或使用隐私模式，确保代码不会存储在外部服务器上
