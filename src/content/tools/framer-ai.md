## Overview

Framer AI is a website builder that uses AI to generate complete websites from text prompts, combining design, copy, and animations.

## Key Features

- **AI Site Generation**: Generate complete websites from text descriptions.
- **Visual Editor**: Drag-and-drop design interface.
- **Animations**: Built-in scroll and interaction animations.
- **CMS**: Content management for blogs and pages.
- **Custom Code**: Add React components and custom code.
- **Hosting**: Fast, global CDN hosting included.

## Pros & Cons

### Pros

- AI generates complete sites quickly
- Beautiful default designs
- Built-in animations
- Fast hosting

### Cons

- Less flexible than Webflow
- Limited export options
- AI output needs refinement
